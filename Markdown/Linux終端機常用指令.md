## 指令連接
> <指令1> && <指令2>

如果 指令1 執行成功則執行 指令2

> <指令1> || <指令2>

如果 指令1 執行失敗則執行 指令2

> <指令1> ; <指令2>

指令1 執行完之後在執行 指令2

> <指令1> | <指令2>

將 指令1 的結果輸出到 指令2

> <指令1> > <檔案路徑>

將 指令1 的結果輸出覆蓋到指定檔案 **(原內容會消失)**

> <指令1> >> <檔案路徑>

將 指令1 的結果輸出接至指定檔案的尾部

## chmod

:::info
Linux的檔案權限可以用 ls/dir -al 看到，
分為可讀r、可寫w、可執行x。
成員設定分為擁有者o、當前群組g、其他用戶o。
:::

:::info
因為在設定權限時使用英文代號並不方便，因此Linux可以將rwx轉為權限數值，r為2分、w為3分，x為4分，總和為7分，因此7分則有完整的控制權。
:::

> chmod <權限/權限數值> <檔案路徑>

更改檔案權限

> example:
>> chmod o=rwx,g=rw,o=- example.txt
>> chmod 750 example.txt
>> 
> 將 example.txt 的權限更改為擁有者++可讀寫執行++ 、當前群組++可讀寫++，其它用戶則++沒有權限++。

## cp
:::info
cp用為於複製，不管是資料夾還是檔案都可以使用此指令複製，並且可以在複制的過程中改名。
:::

> cp <檔案路徑> <檔案路徑2/路徑>

將檔案複製到指定位置

> cp -r <路徑> <路徑2>

將資料夾複製到指定位置

> example:
>> cp example.txt test/example2.txt
>>
> 複製 example.txt 到 test 資料夾底下，並同時改名為example2.txt。
>
>> cp -r test test2
>> 
> 複製 test 資料夾到當前的資料夾底下，並同時改名為 test2。

## find

:::info
找檔案的最佳指令，通常要和 -name 一起使用，也可以使用-iname來忽略大小寫，如此便可以找到特定名稱的檔案。

也可以加 -type 來限定找到檔案的類型，可選目錄d、檔案f。
:::

> find <路徑> -name/-iname <名稱>

在路徑內找到特定名稱的檔案

> find <路徑> -type <d/f> -name/-iname <名稱>

在路徑內找到特定名稱及特定類型的檔案

---

## grep

:::info
在檔案找到特定的字串，可以搭配 -P 使用正則表達式、-i 忽略大小寫、-o 僅輸出符合的字串。
:::

> grep <字串> <檔案路徑>

在檔案內找到特定的字串

> grep -o <正則表達式> <檔案路徑>

在檔案內找到符合正則表達式的字串

### 正則表達式

:::info
正則表達式是為了找到特定格式的字串，基本的格式為[<條件>]。
:::

> [字元]

條件為符合任何在\[\]內的字元
>> a-z、A-Z、0-9
>> 可以直接表達範圍內的字元
>
>> \w
>> 可以用來代替a-zA-Z0-9_

> \{數值\}

代表前面所接的數量

> \.

任意字元

> \*

任意字串

## ls/dir

:::info
最常用的指令之一，因為時常需要查看資料夾的內容，ls和dir的功能相同，皆可加 -al 來顯示隱藏的檔案及詳細資料。
:::

> ls/dir <路徑>

顯示指定路徑的檔案

> ls/dir

顯示目前目錄的檔案

> ls/dir -al

顯示目前目錄的所有檔案連同詳細資料

## rm

:::info
刪除用的指令，可以在後方接路徑或檔案路徑，即可刪除。
如果無法刪除則使用 -rf 來強制執行。
:::

:::danger
-rf 在刪除時並會有任何的提示確認，因此在刪除前請多次確。
:::

> rm <路徑/檔案路徑>

刪除指定的檔案

> rm -rf <路徑/檔案路徑>

強制刪除指定的檔案

---

## strings

:::info
字串查找用，預設可以找到連續4個可視的字元並輸出，也可以和 -n 一起使用，如此便可以指定數量。
:::

> strings <檔案路徑>

在指定的檔案內找到連續4個可視字元並輸出

> strings -n <數量> <檔案路徑>

在指定的檔案內找到連續指定數量個可視字元並輸出

## sudo

:::info
取得root權限，和系統有關的指令都需要使用sudo才能執行，並且需要在第一次的時候輸入密碼。
:::

:::info
如果需要長時間使用root權限，可以使用 sudo -i 進入root終端機。
:::

> sudo <指令>

用root權限執行接方的指令

> sudo -i

進入root終端機
